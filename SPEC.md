# File Browser — 功能规格说明书

## 1. 项目概述

File Browser 是一款自托管的 Web 文件管理器，允许用户在服务器上安装后通过浏览器管理文件和目录，提供类似"私有云盘"的功能。

---

## 2. 功能模块

### 2.1 文件管理

#### 用户故事
- 作为用户，我可以上传文件到服务器指定目录
- 作为用户，我可以下载文件到本地
- 作为用户，我可以创建新文件夹
- 作为用户，我可以重命名文件或文件夹
- 作为用户，我可以删除文件或文件夹
- 作为用户，我可以复制文件或文件夹到其他位置
- 作为用户，我可以查看文件的详细信息（大小、创建时间、修改时间）

#### 验收标准
- [ ] 支持通过拖拽或按钮触发上传文件
- [ ] 支持断点续传（大文件上传中断后可恢复）
- [ ] 下载文件时支持校验和验证
- [ ] 重命名时若文件名冲突，自动添加版本后缀（如 `file (1).txt`）
- [ ] 删除操作需确认，支持批量删除
- [ ] 复制/移动操作可跨目录进行
- [ ] 文件列表按名称、大小、修改时间排序

#### 边界情况
- 上传同名文件时的处理策略
- 上传超出磁盘可用空间时的错误提示
- 删除包含大量文件的目录时的进度反馈
- 网络中断后断点续传的成功率
- 超长文件名或特殊字符（Unicode、空格）的处理
- 嵌套层级过深（>255层）的目录操作

---

### 2.2 文件预览

#### 用户故事
- 作为用户，我可以预览图片（缩略图和大图）
- 作为用户，我可以查看文本文件内容
- 作为用户，我可以预览字幕文件（.srt, .vtt, .ass）
- 作为用户，我可以调整图片预览尺寸

#### 验收标准
- [ ] 图片预览生成缩略图（256x256）和大图（1080x1080）
- [ ] 图片预览支持放大/缩小操作
- [ ] 文本文件预览有行号显示
- [ ] 字幕文件正确解析时间轴
- [ ] 预览加载时显示占位符
- [ ] 支持常见的图片格式（JPG, PNG, GIF, WebP 等）

#### 边界情况
- 超大图片（>50MB）的预览加载策略
- 损坏/不可读图片的降级处理
- 非 UTF-8 编码的文本文件预览
- 二进制文件误点时的提示
- 图片缓存过期策略

---

### 2.3 用户认证与权限

#### 用户故事
- 作为管理员，我可以创建用户账号
- 作为用户，我可以设置个人密码
- 作为管理员，我可以为用户分配不同的权限
- 作为管理员，我可以限制用户只能访问特定目录
- 作为用户，我可以安全地登出系统

#### 验收标准
- [ ] 支持用户名/密码登录
- [ ] 支持多种认证方式：本地认证、代理认证、外部脚本认证
- [ ] 密码强度验证（防止弱密码）
- [ ] 会话过期后自动登出
- [ ] 用户权限细分：管理员、执行、创建、重命名、修改、删除、分享、下载
- [ ] 每个用户可设置独立的作用域（访问路径限制）
- [ ] 隐藏文件规则可配置

#### 权限说明
| 权限 | 描述 |
|------|------|
| Admin | 管理员，可管理所有用户和设置 |
| Execute | 可执行文件 |
| Create | 可创建文件/文件夹 |
| Rename | 可重命名 |
| Modify | 可修改文件内容 |
| Delete | 可删除 |
| Share | 可创建分享链接 |
| Download | 可下载 |

#### 边界情况
- 用户输入错误密码的锁定策略（防暴力破解）
- 会话 token 泄露的检测与处理
- 用户删除自己账号的限制
- 管理员无法被删除时的处理
- 用户权限变更时已有会话的处理

---

### 2.4 文件分享

#### 用户故事
- 作为用户，我可以创建分享链接
- 作为用户，我可以设置分享链接的访问密码
- 作为用户，我可以设置分享链接的过期时间
- 作为用户，我可以撤销已分享的链接

#### 验收标准
- [ ] 生成分享链接后，可复制分享 URL
- [ ] 分享密码使用加密存储
- [ ] 过期时间支持：秒、分钟、小时、天
- [ ] 公开下载接口无需登录
- [ ] 分享列表显示所有已创建的分享

#### 边界情况
- 过期分享链接的访问提示
- 密码错误次数限制
- 分享被删除后链接立即失效
- 大量并发访问同一分享的限流处理

---

### 2.5 搜索功能

#### 用户故事
- 作为用户，我可以实时搜索文件名
- 作为用户，搜索结果根据我的权限过滤

#### 验收标准
- [ ] 搜索框支持实时输入（打字时即时搜索）
- [ ] 搜索结果高亮匹配文字
- [ ] 搜索不区分大小写
- [ ] 搜索结果只显示我有权限访问的文件
- [ ] 支持按路径筛选

#### 边界情况
- 搜索特殊字符（如 `*`, `?`, `.`）的处理
- 大量搜索结果的性能表现
- 空搜索结果的用户提示
- 搜索过程中切换页面的处理

---

### 2.6 访问规则

#### 用户故事
- 作为管理员，我可以设置路径访问规则（允许/禁止）
- 作为管理员，我可以隐藏特定文件

#### 验收标准
- [ ] 支持精确路径匹配规则
- [ ] 支持前缀匹配规则
- [ ] 支持正则表达式规则
- [ ] 可配置是否隐藏点文件（`.` 开头）
- [ ] 规则按优先级排序生效

#### 边界情况
- 规则冲突时的优先级判断
- 正则表达式语法错误的处理
- 规则变更后缓存清理

---

### 2.7 命令执行（钩子）

#### 用户故事
- 作为管理员，我可以配置文件事件触发命令
- 作为管理员，我可以查看命令执行结果

#### 验收标准
- [ ] 支持在文件操作前/后触发命令
- [ ] 支持的事件：删除、上传、复制、重命名、保存
- [ ] 命令可配置为阻塞或非阻塞执行
- [ ] 环境变量传递：文件路径、作用域、触发者、用户名、目标路径

#### 边界情况
- 命令执行超时处理
- 命令执行失败的错误日志
- 恶意命令的防护措施
- 循环触发（命令再次触发同一事件）的防止

---

### 2.8 设置与品牌

#### 用户故事
- 作为管理员，我可以自定义系统名称和Logo
- 作为管理员，我可以自定义主题样式
- 作为管理员，我可以配置上传参数

#### 验收标准
- [ ] 可设置系统显示名称
- [ ] 可上传自定义 Logo
- [ ] 可配置外部链接（帮助文档等）
- [ ] 可自定义 CSS 样式
- [ ] 可配置 TUS 上传的分块大小和重试次数
- [ ] 可设置最小密码长度
- [ ] 登出后重定向页面可配置

#### 边界情况
- 上传非法格式 Logo 的处理
- CSS 注入攻击的防护
- 配置错误导致界面破坏的恢复

---

### 2.9 命令行管理

#### 用户故事
- 作为管理员，我可以通过命令行管理用户
- 作为管理员，我可以通过命令行管理访问规则
- 作为管理员，我可以通过命令行查看/修改配置
- 作为管理员，我可以导出/导入用户数据

#### 验收标准
- [ ] `users` 命令：增删改查用户
- [ ] `rules` 命令：管理访问规则
- [ ] `cmds` 命令：管理命令钩子
- [ ] `config` 命令：查看/修改配置
- [ ] `hash` 命令：生成密码哈希
- [ ] 用户数据支持导出/导入（JSON 格式）

#### 边界情况
- 批量导入用户时的格式校验
- 重复用户名的处理
- 配置修改后服务重启提示

---

## 3. 非功能性需求

### 3.1 性能
- 文件列表加载 < 2 秒（1000 个文件以内）
- 图片缩略图生成后缓存
- 支持大文件（> 1GB）上传

### 3.2 安全
- 所有敏感操作需认证
- 密码加密存储（bcrypt）
- 分享密码加密存储
- 防止路径遍历攻击
- 命令执行默认禁用

### 3.3 可用性
- 响应式设计，支持桌面和移动端
- 中文界面支持
- 键盘快捷键支持
- 拖拽操作支持

---

## 4. 术语表

| 术语 | 说明 |
|------|------|
| 作用域 (Scope) | 用户被限制访问的目录范围 |
| 分享 (Share) | 通过链接共享文件/文件夹的功能 |
| TUS | 断点续传协议，用于大文件上传 |
| 钩子 (Hook) | 文件事件触发自定义命令的机制 |
| Bolt DB | 嵌入式单文件数据库 |

---

## 5. 发布范围（MVP）

第一版发布包含以下核心功能：
- 文件管理（上传、下载、删除、创建、重命名、复制）
- 图片预览与缩略图
- 用户认证与权限
- 文件分享
- 基础搜索
- 命令行用户管理

以下功能作为高级特性：
- 命令执行钩子（安全考虑，默认关闭）
- 正则表达式访问规则
- 自定义品牌和样式
